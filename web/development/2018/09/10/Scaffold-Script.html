Scaffold Script

Sep 10, 2018

Check it out here!

This was done largely as a joke for one of my assignments, I wrote a bash script which generates the linked website. You may note upon observation that the linked website is an instructional guide which lays out how to develop a website scaffold ending in the disclaimer that the user could also develop a bash script which handles that for them…like I did.

A funny thing to note is that I included a link to a random image inserter, Lorem Picsum if you’re curious, in the script, and thus the website it generates will have a random image every time the site is reloaded.

Feel free to download the script and see for yourself!
