Interning with IBM

May 29, 2018

Over the summer of 2018 I carried out an internship with the cryptography division, during which I developed demonstrations of crypto-adapter card functions to be published online. I developed these web demos using C and tested the programs by moving them to a remote server and using Linux command line tools on that server via Putty.

On one occaision I coordinated with one of my co-workers to incorperate code he had developed separately in order to fully complete one of the projects I had been working on, specifically to trigger a pause in the demonstration to show another part of the code. When the time came to submit the code I commited the code to IBM’s custom Git repository, and was added to IBM’s changelogs.
