Magenta

Dec 7, 2018

Take a look at our team website!

The magenta project is one which aims to use the tensorflow framework to utilise machine learning to make better music and art.

I was the manager running a team which sought to apply several upgrades to the existing project. We chose to take on tasks revolving around the Music side of the project, as well as a couple of house keeping fixes.

For example, I personally developped a write up which enabled users to understand the command which generates new music, as well as the commands that allow them to fully control their music generation and train their own models for music patterns.

Another example of some house keeping we did was adding a propper download script to the project, this download script works on Linux OSs and requires existing software already be present on the target machine, however the full download can be completed in around half an hour, rather than however long it could have taken to write and execute every command manually.

We also overlayed a GUI, designing a PHP and CSS web-document which would allow the user to interact with the software without neccessarily having to learn too much command line. This GUI was also critical to our other upgrades to the machine learning, and music generation algorithms themself.
